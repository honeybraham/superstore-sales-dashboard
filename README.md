# superstore-sales-dashboard
Excel Sales Performance Dashboard using Pivot Tables, Charts &amp; Slicers

The Project presents an interactive Excel Dashboard which includes the following:
Power Query Editor
Pivot Tables
Pivot Charts
Slicers
KPI Cards

The dashboard tells about the Superstore Sales by year, region and Product Category.

Key Metrics :
Total Revenue-$9.69M
Total Profit-$1.1M
Profit Margin-11%
Total Order=5496
Average Order Value=$1,763

Dashboard contaians the following featues
Sales vs Profit Trend analysis
Sales By Region
Sales by Year
Sales by Categories
Month Trend Analysis
Filters year,region,categories

Key Insights

2009 recorded the highest sales performance.
Technology category drives maximum revenue.
Western region contributes largest revenue share.
Stable overall profitability at 11% margin

Excel Sheet
